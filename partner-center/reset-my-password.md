# <a name="reset-your-password"></a>重設密碼

如果您嘗試登入合作夥伴中心，而且您不記得密碼，執行下列作業：

1. 輸入您的使用者名稱之後, 在下一個畫面中選取**忘記密碼**。

2. 系統會要求您再次登入。 提供您的電子郵件、 手機或 Skype、，然後選取其中一個選項：

- [建立帳戶]

- 無法存取帳戶

- 登入選項

3. 選取 **無法存取帳戶**。
4. 選取 **工作或學校帳戶**和回答問題。
