# <a name="transition-from-cloud-solution-provider-csp-direct-bill-partner-to-csp-indirect-reseller"></a>從雲端解決方案提供者 (CSP) 直接帳單合作夥伴轉換到雲端解決方案提供者間接經銷商 

>[!Note] 
>這篇文章適用於已決定轉換成間接轉售商直接帳單合作夥伴。 

您可以使用現有的直接帳單的租用戶的間接轉銷商程式中註冊。 

## <a name="get-started"></a>立即開始

1. 請確定您的夥伴設定檔在合作夥伴中心，而目前的 MPN 識別碼。

2. 登入合作夥伴中心全域管理員身分直接帳單租用戶都將其轉換為間接轉銷商。

![概觀](images/direct/direct1.png) 

3. 檢閱您的合作夥伴詳細資料，註冊表單上。

![立即註冊](images/direct/direct2a.png) 

4. 選取立即註冊。 間接轉銷商企業會使用您用於直接業務的相同 AAD 租用戶。

>[!Note]
>一開始，這個新的轉換功能將可供合作伙伴年 9 月到 12 月的週年日期。 如果您不 「 t 有之間年 9 月和年 12 月的週年日，您不會看到這一次的功能。 稍後將收到通知之後 2018 年 12 月的週年日期的合作夥伴，一旦啟用此功能適用於合作夥伴。


5. 您的註冊核准時，重新登入合作夥伴中心。

>[!Note]
> 通常立即核准時，可能需要最多五個工作天。 一旦核准之後，您會收到通知，您已指定在註冊表單中的主要連絡人的電子郵件地址。 您也可以檢查您的註冊狀態，底下**設定**→**夥伴設定**→**夥伴設定檔**→ 節目資訊。 

6. 在您**概觀** 頁面上，您會看到 「 間接轉銷商合約 」。 選取 \[Accept and continue\] (接受並繼續)。 這可讓間接轉銷商功能。

當您接受 「 間接轉銷商合約 」 時，請注意，夥伴設定檔會識別身**兩者**帳單直接和間接轉銷商。

![間接轉銷商合約](images/direct/direct3.png)

>[!Important] 
>一旦註冊為使用新的功能間接轉銷商，沒有任何選項可回復到直接帳單-唯一的租用戶。 請確定您完全評估您的業務需求，再註冊為間接轉銷商。

## <a name="while-you-transition-from-direct-to-indirect-reseller"></a>當您從直接轉換成間接轉銷商

在這個階段，您將會繼續管理您直接客戶的訂用帳戶必須包括計費程序。 您也可以開始接受來自您間接的提供者和間接轉銷商所做的客戶。

![您會同時有直接的帳單和間接轉銷商](images/direct/direct4.png)

## <a name="find-an-indirect-provider"></a>尋找間接提供者

註冊後，間接提供者的連結會出現在左側導覽。 間接轉銷商您將建立關聯性與間接提供者接著可以處理您的計費、 購買的產品為您的客戶和支援基礎結構。

不同間接提供不同支援和服務，因此您應該評估當地提供者，以判斷何者最符合需求。 一般而言，大部分提供者會： 

- 為您提技術訓練與協助
- 協助您行銷產品及服務 
- 管理您的財務和信用額度條款 

搜尋官方清單[Microsoft 間接提供者](https://partnercenter.microsoft.com/partner/find-a-provider)。

瞭解詳細資訊，請閱讀[與間接提供者合作](indirect-reseller-tasks-in-partner-center.md)

## <a name="accept-a-partnership-invitation-from-your-indirect-provider"></a>接受從間接提供者的合作關係邀請

當您找到與間接提供者時，請與間接提供者在合作夥伴中心建立的合作關係。

您選取的間接提供者會傳送給您電子郵件中會帶您前往其邀請合作夥伴中心中的合作關係邀請連結。 請確定您的全域系統管理員登入合作夥伴中心，並遵循邀請連結。 一旦您接受邀請時，提供者的名稱會出現在 間接提供者清單中。

## <a name="acquire-new-customers-as-indirect-reseller"></a>間接轉銷商以取得新客戶

您和您間接提供者必須將轉銷商與客戶的關聯性。 這可讓您管理客戶的訂用帳戶和其代表的服務。 若要取得新的客戶現有的 Azure AD 租用戶的人員，您可以邀請客戶與您和您的提供者建立的轉銷商關係，在相同的時間。 

若要建立間接轉銷商邀請：

1. 選取 **間接提供者**合作夥伴中心從左導覽 

2. 選取 **\[邀請新客戶\]** 以邀請客戶同時與您及間接提供者建立經銷商關係。 提供者必須有與您的客戶轉銷商關係，因此當客戶想要購買新的訂用帳戶，或將新的授權新增至現有的訂用帳戶，他們就可以提交訂單，您的客戶代表。

3. 在下一頁，檢閱草稿電子郵件訊息。 您可以開啟電子郵件、 草稿訊息，或者您可以將訊息複製到剪貼簿，並將它貼到電子郵件。

4. 編輯要說什麼需要但一定要包含的連結，因為它個人化的客戶直接連接到您的帳戶和您的提供者帳戶的電子郵件中的文字。 然後選取**完成**。

5. 客戶授權您和您的提供者成為其記錄可查經銷商之後，您就會有系統管理員權限，可代表他們管理其訂閱、授權和使用者，並且間接提供者可代表他們提交訂單。
6. 若要管理客戶的帳戶、服務、使用者及授權，請選取客戶名稱旁邊的向下箭頭以展開其記錄。 

不同於直接帳單合作夥伴間接轉售商無法在合作夥伴中心建立新客戶的 Azure AD 租用戶。 您的提供者會建立租用戶，並指定您為這位客戶的間接轉銷商。 這可確保客戶會出現在您的客戶清單，在合作夥伴中心。

>[!Note]
>您無法使用直接帳單功能為您為間接轉銷商取得的客戶建立購買項目。

4. 編輯要說什麼需要但一定要包含的連結，因為它個人化的客戶直接連接到您的帳戶和您的提供者帳戶的電子郵件中的文字。 然後選取**完成**。

## <a name="managing-your-direct-bill-customers-and-your-indirect-reseller-customers"></a>管理您的客戶直接的計費和間接轉銷商客戶

您以不同的方式管理您的客戶直接的計費和間接轉銷商客戶。 

**直接向客戶計費 （您不會執行為間接轉銷商的項目）**

- 建立產品的訂單
- 管理 Azure 保留的項目
- 管理其訂單歷程記錄
- 購買軟體
- 直接向客戶收取費用

**間接轉銷商客戶**

- 間接提供者為您的客戶訂單的產品
- 管理客戶的授權及使用者
- 處理訂用帳戶續約

**若要識別您為直接帳單協力廠商取得的客戶**

1. 選取**客戶**

2. 選取客戶以檢視其詳細資料

3. 如果其中一個您身為直接帳單合作夥伴取得此客戶，您會看到選項**新增**或是**檢視產品**，您會看到其訂用帳戶。

4. 如果客戶有與您的間接轉銷商關聯性，將無法使用這些選項。

## <a name="what-happens-when-your-direct-bill-capabilities-have-been-restricted"></a>當您直接帳單的功能已限制時，會發生什麼事

您已決定轉為間接轉銷商，且有雙重的註冊，兩者都直接帳單和間接轉銷商，或您沒有進行明確決定尚未註冊為間接轉銷商，是否直接帳單合作夥伴不符合新的需求csp 直接帳單合作夥伴計劃就會通知 microsoft 其直接帳單功能就會受到限制時。 

在您**夥伴設定檔**下方**程式設計資訊**您會看到您的 Microsoft 雲端解決方案提供者狀態已變更為**限制**。

## <a name="affected-direct-bill-capabilities"></a>直接的帳單功能會受到影響

當您直接帳單的功能已限制時，您可以不再進行新購買，為您的客戶在合作夥伴中心。 這包括： 

- Azure 訂用帳戶
- 以授權為基礎的訂用帳戶
- 將新的附加元件加入至現有的授權為基礎訂用帳戶
- 進行一次購買的軟體和保留項目產品，例如軟體訂用帳戶、 永久的軟體，以及 Azure 保留的虛擬機器執行個體。

您也無法供自己使用，使用 Azure 合作夥伴共用的服務供應項目，CSP 方案購買新的 Azure 訂用帳戶。 

除了新的購買，您無法存取下列直接帳單功能在合作夥伴中心： 
- 您無法建立新的客戶租用戶。 **建立客戶**選項底下**客戶**在合作夥伴中心內將無法使用。 

- 您無法產生對客戶要求的直接轉銷商關聯性的邀請。 **要求轉銷商關係**選項底下**客戶**在合作夥伴中心內將無法使用。 

不會影響現有直接帳單的訂用帳戶。 它們會維持有效，並會自動更新。 您將繼續由 Microsoft 直接計費，直到它們取消。 您還是可以透過下列方式來管理現有的訂用帳戶： 

- 暫停現有的訂用帳戶 

- 調整以授權為基礎的現有訂閱的基座數目 

- 調整現有的附加元件訂用帳戶的基座數目。 注意：您 ccan't 會將新的附加元件新增至現有的訂用帳戶中，因為它們會被視為新購買。 

- 部署新的 Azure 資源及管理現有的 Azure 訂用帳戶下現有的 Azure 資源。 這包括透過 Azure marketplace 和 Visual Studio 訂用帳戶，您可以使用哪些產品。 

>[!Note] 
>如果您已經註冊為間接轉銷商直接帳單合作夥伴租用戶，您就能夠傳送邀請給客戶要求的間接轉銷商關聯性，而不是直接的帳單。 

- 您無法建立新的 「 沙箱 」 租用戶。 每個直接帳單合作夥伴租用戶可以建立一個沙箱租用戶，以直接帳單 API 整合。 如果您先前未建立一個，將無法這樣做之後您將會導向帳單協力廠商功能已受到限制。 

## <a name="enroll-for-indirect-reseller-incentives"></a>註冊間接轉銷商獎勵 

您已成功在您現有的直接帳單合作夥伴租用戶上註冊為間接轉銷商之後，您會收到邀請，以間接轉售商獎勵註冊 30 天內。 邀請為基礎的夥伴目前與您的 CSP 合作夥伴租用戶相關聯的 MPN 帳戶。 與合作夥伴的 MPN 帳戶相關聯的電子郵件地址，就會傳送邀請。 

您還有資格以註冊該相同的合作夥伴租用戶直接帳單獎勵方案。 您必須個別管理的應用程式。

另請參閱[其他有關成為間接轉銷商](https://assetsprod.microsoft.com/csp-directbill-to-indirect-transition.pdf)而[間接轉銷商直接帳單常見問題集](https://assetsprod.microsoft.com/mpn/direct-bill-partner-faq.pdf)。
