# <a name="billing-basics"></a>帳單基本知識

**適用對象：**

-  合作夥伴中心
-  美國政府適用的 Microsoft Cloud 合作夥伴中心
-  Microsoft Cloud 德國合作夥伴中心

## <a name="overview-of-billing"></a>帳單概觀
除了每月計費選項之外，您現在還有年度計費選項。 如需詳細資訊，請參閱[新計費功能的常見問題集](faq-about-new-billing-features.md)。

### <a name="dates"></a>日期

您的每月帳單日期為您在註冊期間選取的日期。 Microsoft 將會傳送包含您帳單日期的確認電子郵件。

您可以在一 (1) 個月前事先看到價目表，價目表每個月會更新一次。 授權型價格保證在訂閱期間維持不變，通常是從購買日期算起的 12 個月。 用量型價格每個月可能會有所變更。 價格如有任何變更，我們將會透過我們的 \[合作夥伴價目表\] 發佈，並在 30 天前通知。


### <a name="payment-terms"></a>付款條款

付款條款所淨 60 天。 付款必須在發票到期日 (帳單日期 60 天後) 之前完成，否則帳戶將會成為到期未付的狀態。

到期未付的帳戶將會被暫停或終止參與雲端解決方案提供者計畫。 暫停的帳戶將無法建立新客戶或訂單、要求經銷商關係、提高或降低訂閱數量、訂購附加元件訂閱、轉換或切換訂閱，並將會被限制只能管理現有的客戶、訂閱及資源，直到帳戶將拖欠款項付清為止。 合作夥伴在支付未付帳單之後，便能重新取得暫停帳戶的完整功能。


### <a name="billing-rules"></a>帳單規則
<ul>
<li>您每個月會收到一份雲端解決方案提供者計畫的發票。</li>

<li>授權型訂閱是根據購買的授權數目，而非使用的授權數目收費。</li>

<li>Azure (用量型訂閱) 是根據計量費率，依使用量收費。</li>

<li>訂閱期間的價格保證不會變動。 但價格可能會在訂閱續約時有所變更。</li>
</ul>

### <a name="invoice-availability"></a>發票可用性

您可以在合作夥伴中心的 \[帳務\] 頁面，檢視和下載您的發票和對帳檔案。 請注意，每月發票於您所選帳單日期的 4 天內，可在合作夥伴中心中查看。


### <a name="adjustmentscreditscancellations"></a>調整/點數/取消

計點或調整之後，您會在下一次計費發票上看到延遲的調整與點數。


### <a name="tax"></a>稅金

我們將會根據您的 (而非您客戶的) 詳細資訊做為 Microsoft 與您之間的帳務關係收取稅金。 您可以在登入期間或透過服務要求，提交您的統一編號。 您會在下一個帳單週期看到反映的變更。

如需<strong>預扣和減免營業稅</strong>，您必須透過服務要求提交稅務文件。 您會在下一個帳單週期看到變更與退款。

如需<strong>減免加值稅 (VAT)</strong>，您必須透過服務要求提交您的加值稅識別碼 (經過 Microsoft 驗證)。 您會在下一個帳單週期看到變更與退款。

請向您當地的稅務機關或稅務顧問進一步了解稅務詳細資料。

 
## <a name="license-based-billing"></a>授權型帳單

### <a name="billing-frequency"></a>帳單頻率

當您購買訂閱，以代表客戶時，您可以選擇每月或年度計費。 如果您想要在稍後變更您的帳單週期，使用下列程序。 

如果您有許多每月計費的訂閱，以及您想要與常見的帳單日期一致它們，從每月計費切換為年度計費非常有用。 從年度計費切換至每月計費中非常有用量身訂做那些個別客戶您帳單日期。 

當您考慮以變更您的帳單週期，請記住下列各項： 

-   當您變更了帳單週期時，年度一詞會更新以反映您變更了帳單週期的日期和新的續約日期建立。 

-   您可以變更只有一詞年度訂閱;您無法變更試用訂用帳戶的頻率。 

-   如果沒有共用的順序 （例如，父項訂閱） 的訂閱，所有的那些訂閱將會更新。 

-   不執行促銷，套用新的詞彙。

-   定價將會套用變更時，不繼承自上一個字詞次。

-   共用順序的所有訂閱都必須是使用中狀態。

-   您可以變更為您要的次數的帳單週期。 


<b>若要變更訂閱的帳單週期</b>
<ol>
<li>從合作夥伴中心功能表中選取<b>客戶</b>，然後選取您想要變更訂閱的客戶。
<li>在客戶的<b>訂閱</b>頁面上，選取您想要變更的訂閱。 
<li>在詳細資料頁面上下<b>帳單週期</b>，, 選取<b>每月</b>或<b>年度</b>。 您會看到確認頁面變更帳單週期，以及一份即將變更訂閱的相關的重要資訊。 
<li>選取<b>[確定</b>] 進行變更，或<b>取消</b>復原它。 
</ol>

### <a name="billing-rules"></a>帳單規則
<ul>
<li>訂閱是以年為單位，而且會自動續約。</li>
<li>帳單是 12 個月付款或每年度訂閱的一個年度付款。</li>
<li>您若使用授權型服務，則必須以前一個帳單週期結束時的授權數目為根據，事先為下一個帳單週期付費。</li>
<li>如果授權數目有變更，您會於期後被收取費用/獲得點數 (根據授權-日數按比例計算)。 按比例計算使用下列公式：\[ROUND((ROUND(單價 * 數量 / 日數在該月中的比例, 2) * 按比例計算的日數) / 數量, 2) * 數量\]</li>
<li>費用是根據售出的基座數目 (而非佈建的基座數目) 計算。</li>
</ul>

### <a name="adjustmentscreditscancellations"></a>調整/點數/取消

取消授權型服務目前不收取提早終止費用。

授權型服務的取消點數是針對期中取消，按照未使用日數的比例計算 (根據上述公式以及減少的授權數目)。
 

## <a name="usage-based-billing"></a>用量型帳單

Azure 是採取「隨付隨用」模式，有使用 Azure 服務，才需要付費。

### <a name="billing-rules"></a>帳單規則
<ul>
<li>訂閱是以月為單位，並會以新的計量服務費率自動續約。 帳單是根據用量於每個月期後付款。</li>
<li>計量服務的費率可能會在發票週期內有所變更。
<ul>
<li>價格提升：會在 30 天前提供通知。</li>
<li>價格降低：會在變更當天反映。</li>
<li>現有訂閱會使用帳單週期一開始的實際費率。</li>
<li>新訂閱 (於帳單週期內建立) 會使用建立日期的實際費率。</li>
</ul></li>
</ul>

### <a name="adjustmentscreditscancellations"></a>調整/點數/取消

您會在下一個月的對帳發票上看到調整後的付款。

取消用量型服務目前不收取提早終止費用。

您會在下一個月的對帳發票上看到任何類型的點數，包括 SLA 點數。


## <a name="one-time-billing"></a>一次性帳單

雲端解決方案提供者 (CSP) 中的合作夥伴可提供 Microsoft Azure 虛擬機器上的保留執行個體給客戶。 客戶可事先保留虛擬機器 – 使用一年或三年期限 – 並體驗 Azure 使用率的大幅度節省。   

您的客戶需事先支付 Azure 保留的 VM 執行個體。 當您代表客戶購買 Azure 保留的 VM 執行個體時，您會收到這些一次性費用的發票和對帳檔案。 

>[!IMPORTANT]
>如果您在所用貨幣與您不同的市場中為客戶購買 Azure 保留的 VM 執行個體，預設帳單貨幣會以客戶的市場為準，而不是您的所在位置。 如果您在多個市場擁有客戶，您會針對需要付費客戶的每種貨幣收到不同的發票和對帳檔案，讓您可用適當的貨幣向客戶開立發票。 

若要存取一次收費發票和對帳檔案中，選取**帳單**從合作夥伴中心，然後選取**一次**。 

### <a name="manage-your-one-time-billing"></a>管理您的一次性帳單

#### <a name="view-your-current-billing-status-invoices-and-recon-files"></a>檢視您目前的帳單狀態、發票，以及對帳檔案

1.  在合作夥伴中心，選取**帳單**，然後**一次**，檢視您的帳單狀態。 
2.  選取發票或對帳檔案來檢視更多詳細資訊。 

#### <a name="view-a-customers-order-history"></a>檢視客戶的訂購記錄

1.  從合作夥伴中心功能表中選取**客戶**。
2.  在 **\[客戶\]** 頁面上，尋找您要檢視其訂購記錄的客戶，然後選取向下箭號來展開客戶的記錄。 
3.  選取 **\[檢視訂單\]** 來顯示訂購記錄。

#### <a name="download-a-credit-note"></a>下載貸記金額附註

您練習完整的貸記金額 （Void 發票） 和重新計費的事件中，我們將會提供您看原始發票，以及完整新的文件取消貸記金額附註。 您可以要求貸記金額/重新計費，原因如下：
-   地址或購買順序校正
-   產生發票，再套用稅金退款。 您可以要求貸記金額/重新計費取得提取回至原始發票稅金退款。 這也適用於退款，因為您可以要求貸記金額/重新計費的原始的發票，然後提取中退款。
