# <a name="billing-overview"></a>帳單概觀

**適用於**

- 合作夥伴中心
- 雲端解決方案提供者 (CSP) 計畫中的合作夥伴

**適當的角色**

- 系統管理代理人
- 帳單系統管理員
- 全域系統管理員
- 技術服務代理人
- 銷售代理人

## <a name="find-your-bill"></a>尋找帳單

尋找您的帳單：

1. [登入合作夥伴中心儀表板](https://partner.microsoft.com/dashboard/home)。

2. 在左側功能表中，選擇 [帳單]  。

3. 在 [帳單]  頁面上，您可以下載最新的帳單，或在 [帳單記錄]  區段中下載先前的帳單。

## <a name="bill-your-customers"></a>向客戶計費

Microsoft 對於您開立發票的處理方式上沒有任何要求或條件。

若要判斷客戶的使用量，請[查看您的對帳檔案](#find-your-bill)。 您可以使用客戶名稱和其他相關欄位來決定使用量。

接下來，針對您所提供的產品與服務自行增加費用。 然後，將包含所有費用的單一帳單傳送給客戶。

### <a name="billing-types"></a>帳單類型

合作夥伴中心內的帳單類型包含**授權型帳單**、**用量型帳單**、**一次性帳單**。 如需詳細資訊，請參閱[合作夥伴中心的帳單類型](billing-different-types.md)完整說明。

### <a name="billing-currency"></a>計費貨幣

您將以您所在國家或地區的貨幣來支付產品的費用。 無論您銷售產品的客戶位置在哪裡，您都必須支付相同的費用。

## <a name="invoices"></a>發票

您的發票是目前計費週期內所有費用的摘要。 這涵蓋整個程式、所有產品、所有客戶的費用。 如需每月和年度計費案例的範例，請參閱[常見的計費案例](common-billing-scenarios.md)。

您的發票會在您所選計費日期 (UTC 時間) 的兩 (2) 天後提供。 例如，如果您有 9 月 12 日帳單日期，則發票產生程序將從 13 日 12:00AM UTC 開始，並在 14 日 12:00AM UTC 完成。 如果您在 15 日下午 11:59 UTC 沒有看到您的發票，表示您不在服務等級協定中，而且應該提出服務要求。

如果雲端解決方案提供者 (CSP) 計畫中的合作夥伴選擇每月計費，應於 60 天內，向 Microsoft 支付其客戶的訂閱費用 (包括授權型與用量型)。

## <a name="price-lists"></a>價目表

價目表每個月更新一次。 預覽的價目表會在一個月前預先提供。

檢視價目表：

1. [登入合作夥伴中心儀表板](https://partner.microsoft.com/dashboard/home)。
2. 在左側功能表中，選擇 [銷售]  ，然後選取 [定價與供應項目]  。
3. 在 [定價與供應項目]  頁面上，您可以看到 6 個月的價目表 (包括當月) 和預覽價目表 (如果有)。

**授權型**價格保證在訂閱期間維持不變，通常是從購買日期算起的 12 個月。 

**用量型**價格每個月可能會有所變動。

**產品、服務、軟體訂閱**的價格在訂閱期間為保證價格。 不過，當您續訂時，價格可能會改變。

計點或調整之後，您會在下一次計費發票上看到延遲的**調整**與**點數**。

## <a name="payment-terms"></a>付款條款

付款期限為 60 天整。 付款必須在發票到期日 (帳單日期 60 天後) 之前完成，否則帳戶將會成為到期未付的狀態，並影響您在 CSP 的註冊。 

在您支付逾期金額之後，便能重新取得暫停帳戶的完整功能。

### <a name="taxes-and-vat"></a>稅金和加值稅

我們將會根據您的 (而非您客戶的) 詳細資訊收取稅金，因為帳務關係是在 Microsoft 與您之間。 您可以在帳戶設定期間提交您的稅務識別碼，或稍後透過支援要求提交。 您會在下一個帳單週期看到反映的變更。

如需**預扣和減免營業稅**，您必須透過支援要求提交稅務文件。 您會在下一個帳單週期看到變更與退款。

如需**減免加值稅 (VAT)** ，您必須透過服務要求提交您的加值稅識別碼 (經過 Microsoft 驗證)。 您會在下一個帳單週期看到變更與退款。

請向您當地的稅務機關或稅務顧問進一步了解稅務詳細資料。

### <a name="annual-billing-rules"></a>年度計費規則

訂閱是以年為單位，而且會自動續約。

每個年度的訂閱，帳單是每個月付款一次，共計 12 次，或一次年度付款。

您若使用授權型服務，則必須以前一個帳單週期結束時的授權數目為根據，事先為下一個帳單週期付費。

如果授權數目有變更，您會於期後被收取費用/獲得點數 (根據授權-日數按比例計算)。 按比例計算使用下列公式：

`[ROUND((ROUND(Unit Price * Quantity / Number of days in pro-rated Month, 2) * Number of pro-rated days) / Quantity, 2) * Quantity]`

費用是根據售出的授權數目計算，而非佈建的授權數目。

### <a name="change-billing-frequency"></a>變更帳單週期

變更客戶的線上服務計費週期：

1. [登入合作夥伴中心儀表板](https://partner.microsoft.com/dashboard/home)。
2. 在左側功能表中，選擇 [客戶]  。
3. 選取您要變更訂閱帳單週期的客戶。
4. 在客戶的 [訂閱]  頁面中，選取要變更的訂閱。
5. 在 [詳細資料]  頁面的 [帳單週期]  下，選取 [每月]  或 [每年]  。 您會看到確認頁面，其中包含有關變更帳單週期的重要資訊，以及即將變更的訂閱清單。
6. 選擇 [確定]  進行變更 (或 [取消]  復原)。

### <a name="adjustmentscreditscancellations"></a>調整/點數/取消

若您取消授權型服務，Microsoft 不會收取提早終止費用。

授權型服務的取消點數是針對期中取消，按照未使用日數的比例計算 (根據上述公式以及減少的授權數目)。

### <a name="billing-rules"></a>帳單規則

訂閱是以月為單位，並會以新的計量服務費率自動續約。 每個月會依據您上個月的使用量向您收取費用。

計量服務的費率可能會在發票週期內有所變更。

#### <a name="price-changes"></a>價格變更

當**價格提升**時，會在 30 天前通知。

若**價格降低**，則會反映在變更當日。

**現有訂閱**會使用計費週期一開始的實際費率。

於同一計費週期內建立的**新訂閱**，會使用建立日期的實際費率。 

### <a name="adjustmentscreditscancellations"></a>調整/點數/取消

您會在下一個月的對帳發票上看到調整後的付款。

若您取消用量型服務，Microsoft 不會收取提早終止費用。 

您會在下一個月的對帳發票上看到任何類型的點數，包括 SLA 點數。

>[!IMPORTANT]
>如果您為客戶購買 Azure 保留和/或軟體訂閱，而客戶所在位置的貨幣與您不同，預設帳單貨幣會以客戶的位置為準，而不是您的所在位置。 如果您在多個不同位置的客戶，您會針對需要付費客戶的每種貨幣收到不同的發票和對帳檔案，讓您可用適當的貨幣向客戶開立發票。

## <a name="manage-one-time-billing"></a>管理一次性帳單

### <a name="billing-status-invoices-and-reconciliation-files"></a>帳單狀態、發票和對帳檔案

檢視您目前的帳單狀態、發票、對帳檔案：

1. [登入合作夥伴中心儀表板](https://partner.microsoft.com/dashboard/home)。
2. 在左側功能表中，選擇 [帳單]  ，然後選取 [一次性]  。
3. 在帳單狀態頁面中，選取發票或對帳檔案來檢視更多詳細資訊。

### <a name="customer-order-history"></a>客戶訂購記錄

檢視客戶的訂購記錄：

1. [登入合作夥伴中心儀表板](https://partner.microsoft.com/dashboard/home)。
2. 在左側功能表中，選擇 [客戶]  。
3. 在 [客戶]  頁面上，尋找您要檢視訂購記錄的客戶。 選取向下箭號以展開客戶的記錄。
4. 選取 [檢視訂單]  來顯示客戶的訂購記錄。

### <a name="credit-notes"></a>欠款單據

您可能需要出於以下原因申請抵免/重新計費：

- 您需要更正地址或採購單。
- 在發票產生之後套用退稅。 您可以要求抵免或重新計費以將退稅退回原始發票。 這也適用於退款。 您可以申請原始發票的抵免/重新計費，然後引進退款。

如果您申請抵免或重新計費，我們會提供您**欠款單據**以取消原始發票。
