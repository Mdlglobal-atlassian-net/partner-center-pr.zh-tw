# <a name="view-your-incentive-payments-and-programs"></a>查看您的獎勵付款和計畫

**適用於**

-  夥伴中心

**適當的角色**

-  獎勵系統管理員
-  獎勵使用者 
-  獎勵唯讀檢視器
-  全域系統管理員
-  MPN 合作夥伴系統管理員

使用這些頁面來查看過去、暫止和即將到來的付款和收益、每個位置中的獎勵計畫狀態，以及完整的交易歷程記錄的詳細資料。 您也可以下載或匯出任何交易的詳細資訊。 所有獎勵頁面都會顯示美元的金額。

>[!NOTE]
>您只會看到與您相關聯的 MPN 識別碼和程式的資料。 如果您想要存取其他資料，請參閱您的獎勵系統管理員。 

## <a name="access-the-incentives-pages"></a>存取獎勵頁面

您可以透過下列幾種方式從合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)存取獎勵資訊：

- 從左側功能表中選取 [**獎勵**]。 [**獎勵**] 底下會出現數個選項，包括 **[總覽**]。 探索這些選項，以瞭解獎勵計畫、獎勵付款、收益和索賠的狀態。
- 選取右上角的 [ **Money** ] （支出）圖示。 然後，您可以選取其他選項，例如 **[交易歷程記錄**]、[**付款**]、[**匯出資料**] 或 [ **Office 銷售報告**]。

下列頁面將在此說明：

- 我的獎勵總覽頁面
- 付款頁面
- 交易記錄頁面
- 匯出資料頁面和其他相關聯的匯出頁面

>[!NOTE]
>如需合作夥伴中心的獎勵和獎勵功能的詳細資訊，請參閱[合作夥伴投資和獎勵](https://partner.microsoft.com/membership/partner-incentives)（需要登入）。

## <a name="my-incentives-overview-page"></a>我的獎勵總覽頁面

**我的獎勵總覽**會顯示獎勵計畫的整體狀態，以及每個程式在每個位置的狀態。 這項資訊包括您在獎勵計畫中註冊後的獎勵總收益和付款。

此頁面也包含收益和付款（依位置或程式），以及您可能需要採取的任何進一步動作，以便在特定位置註冊程式。

若要存取**我的獎勵總覽**：
 
1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。
2. 從功能表中選取 [**獎勵**]，然後選取 **[總覽**]。
3. 在頁面頂端查看 [收益] 和 [付款] 摘要，並在下表中取得進一步的詳細資料。 您也可以排序、分組和展開隨附的資料表：

   - 若要依資料行排序，請選取資料行名稱。
   - 若要依程式分組，請選取資料表上方的 [**依程式**] 索引標籤。
   - 若要依位置分組，請選取資料表上方的 [**依位置**] 索引標籤。
   - 若要查看特定群組內註冊的詳細資料，請選取位於指定資料列結尾的 v 形符號。 這會展開您的視圖。

4. 如果需要進一步動作以在程式中註冊，此資訊將會出現在 [**狀態**] 欄中。 在此情況下，請選取 v 形符號，以瞭解您需要採取的後續步驟。

>[!NOTE]
>請參閱下表，以深入瞭解 [**狀態**] 欄中顯示的不同註冊狀態。


| **狀態**         | **出現的時機** |
|:------------------------------------|:------------------|
| 需要採取動作  | 合作夥伴已接受邀請來註冊獎勵計畫，但可能需要更新銀行或稅務資訊。 如需後續步驟或連結以在合作夥伴中心更新您的銀行或稅務資訊，請參閱**必要動作**一欄。 |
| 已停用  | 獎勵系統不再提供特定的獎勵計畫。 |
| 過  | 已驗證所有稅務和銀行資訊。 合作夥伴不需要進一步的註冊動作。 |
| 加入  | 使用者不是獎勵系統管理員，而是註冊**所需的動作**或**驗證註冊**狀態。|
| 非使用中/不合格 | 獎勵計畫目前可能不會開放註冊，或者合作夥伴不符合目前註冊或重新註冊的資格。 <br> 如果狀態不**合格**，表示合作夥伴不符合程式目前的資格需求;按一下 [註冊狀態] 底下的 [**查看資格需求**] 連結，將會顯示資格的需求，以及符合哪些需求。 |
| 已邀請  | 新的獎勵方案註冊邀請已傳送給合作夥伴，但合作夥伴尚未開始註冊程式。 連續的 [**必要動作**] 資料行會顯示後續步驟和任何相關的連結。  |
| 正在驗證註冊  | 合作夥伴已經完成或更新了新的或現有註冊的銀行和稅務資訊，正在等候 Microsoft 驗證此資訊。 在驗證過程中，**驗證註冊**最多可能會出現48小時。  |

## <a name="payments-page"></a>付款頁面
此頁面上的總計代表您所使用的所有 MPN 識別碼。 您可以依 [參與者識別碼]、[程式]、[付款識別碼] 和 [賺取類型] 進行篩選。 金額會以美元為單位來提供。 付費值也會以 [付費貨幣] 顯示。 

|**領域**   |**描述**    |
|------------------|:-------------------------------------|
|今年總付費        |所有 MPN 識別碼的總金額，以美元為單位向您收費。                                      |
|下一個估計付款      |您的單一下一期付款（即使有其他人即將推出），以美元為單位。                                     |
|上次付款           |您最近付款的數量（以美元為單位）、計畫名稱和 MPN 識別碼。                                      |
|依來源的付款       |過去12個月內由程式代表的付款金額（以美元為單位）。                                      |
|付款                       |選取 [**付費**] 或 [**暫**止]，然後依您想要的方式排序。 如需特定付款的其他詳細資料，請選取 [**查看**]。 若要下載付款匯款語句的複本，請選取 [**下載**]。 請注意，交易記錄資料最多可能需要24小時的時間才會出現，因此您可能不會立即看到相關的收益。  |

若要匯出此頁面上的任何資料，請選取 [**匯出**]，然後依照 [匯出資料] 頁面上的指示進行。 

## <a name="transaction-history-page"></a>交易記錄頁面
此頁面會顯示您所有的個人收益，包括每個的日期、類型和收益。 您可以選取要查看的時間週期，也可以依註冊識別碼、程式、付款識別碼、賺取類型、杠杆和狀態進行篩選。 資料適用于目前的會計年度（7月1日至6月30日）和前兩個會計年度。 

若要查看收益的更多詳細資料，請選取頁面右側的向下箭號。 這會顯示杠杆、收益金額、產品和客戶。 如果因為某些原因而無法使用任何資料，但您需要存取它，請聯絡支援人員。 如果收益是調整的結果，而不是交易，則不會顯示 [產品] 和 [客戶] 欄位。 

若要匯出此頁面上的任何交易資料，請選取 [**匯出**]，然後依照 [匯出資料] 頁面上的指示進行。 從 [交易記錄] 頁面匯出的檔案會顯示交易貨幣中的資料、交易貨幣和美元的收益，以及付費貨幣值。

## <a name="export-data-page"></a>[匯出資料] 頁面
遵循此頁面上的指示，匯出您想要的資料。 

**附註：**
-   您只會看到與您相關聯的 MPN 識別碼和程式的資料。 如果您想要存取其他資料，請參閱您的獎勵系統管理員。 
-   [匯出資料] 頁面不會自行重新整理。 您可能需要手動重新整理頁面，才能看到最新的資料。 
-   您的篩選可能會導致**資料無法使用**錯誤。 這可能表示您已保留在三個月內選取的預設時間週期，然後從該期間外的收益中選取付款識別碼。 請展開您的時間週期，然後再試一次。
