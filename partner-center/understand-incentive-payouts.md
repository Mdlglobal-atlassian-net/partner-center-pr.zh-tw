#<a name="understand-your-incentive-payouts"></a>了解您的獎勵支付

**適用於**

-  合作夥伴中心

獎勵系統管理員及使用者會收到他們自其獎勵計畫獲得到或即將獲得之收益與支出的概觀。   

##<a name="summary-information-on-earnings-and-payout"></a>收益和支付摘要資訊

獎勵使用者和獎勵系統管理員可以透過 **\[支付摘要\]** 查看他們在其所參與之獎勵計畫中的財務狀況。 您可以視需要詳細分析每個部分的摘要資訊。 

在摘要頁面上，您可以追蹤分析您在哪幾個月獲得付款以及對應計畫類型的趨勢。 另一方面，您也可以依計畫和依位置查看您賺取的收益。 這可讓您了解您獲致成功之處，以及可投入更多精力來行銷和銷售的地方。 [支付摘要] 頁面會顯示收益和支付，獎勵資料每 48 小時重新整理一次。 圖表和表格會反映最新的資訊。 

摘要頁面使用的主要指標有三種：

|**主要指標**   |**定義**   |
|---------------------|:-----------------|
|至今已付總額|Microsoft 已支付給您的金額|
|未付收益|您有賺取但尚未排定支付的金額|
|近期預定付款| 已排定支付的金額|


分析每項獎勵計畫的收益和支付。 追蹤收益和支付之間的時間表，對規劃未來資源需求及其他事項十分有用。 您還可以發現哪些位置在獎勵計畫中更成功或更活絡。 

## <a name="payment-trends"></a>付款趨勢

摘要的 [付款趨勢] 區段顯示依據您所參加計畫細分的支付。 付款趨勢檢視使用頁面上的資料篩選器進行控制。 付款排程取決於計畫設計。 如需詳細資訊，請參閱您的計畫指南。 

## <a name="total-earnings-map"></a>總收益地圖

總收益地圖顯示您已在哪些地點獲利。 您進行交易的位置會反映在地圖中。 這表示，如果您在美國進行銷售，就會在地圖中看到結果。 這個檢視與計畫無關。 

## <a name="product-earnings-summary"></a>產品收益摘要

產品/計畫收益和付款表按照每項計畫/產品顯示總收益金額、未付收益、近期預定付款及至今已付總額的明細。 



## <a name="analyze-incentives-earnings"></a>分析獎勵收益

了解您的組織還可以在什麼地方透過獎勵計畫賺取更多收益，以及您的哪些收益已獲付款。 這項資料可以釐清您賺取收益但尚未獲支付的來源。  了解哪些計畫和活動已取得最大成果，以及查明您的哪些地點正在獲利。 您可以依據計畫查看總計潛在收益的結果、您可參與計畫的時間以及該計畫到期的時間。 收益的每項計畫及類型都會有不同的需求以及開始和結束日期。 

## <a name="earning-trends"></a>收益趨勢 

收益趨勢圖表顯示您收益的一般發展或變動方向。 圖形檢視 (可透過日期範圍選取器來控制) 將會根據您選取的日期範圍顯示資料。 

### <a name="earning-trends-category-definitions"></a>收益趨勢分類定義

|   |**定義**   |
|----------------|:--------------|
|已付收益|已順利支付的收益|
|未付收益|尚未排定付款的收益|

### <a name="earning-types"></a>收益類型

有兩種收益類型：

- 回贈
- 調整

## <a name="unpaid-earnings-summary"></a>未付收益摘要

未付收益表顯示尚未排定付款之未付收益的摘要。 這是依據計畫/產品和收益類型來劃分。 您可以查看收益期間何時開始以及收益最後確定下來的結束日期。 收益已決待付日期是收益要求條件已完備，足以符合獲得支付資格的日期。 如果收益可能會過期，不予支付時，則適用到期日。

## <a name="payments-and-statements"></a>付款與對帳單

[付款與對帳單] 頁面上的地圖向您顯示根據位置區分的交易付款。 例如，如果您在美國進行銷售並收到付款，這個情況會在地圖上顯示。 這個檢視是您所參與之所有計畫的彙總。

## <a name="payments"></a>付款

付款表顯示過去 12 個月中進行的所有付款。 這會依據計畫和收益類型來細分。 您可以查看收益期間開始時間，以及收益完成的時間。 當付款匯到銀行時，您會看到付款日期和付款識別碼。

## <a name="statements"></a>對帳單

對帳單會在支付款項匯到銀行之後產生。 按一下 **\[檢視\]** 即可查看支付詳細資料。 

[下載交易] 會顯示與該月份所進行支付相關聯之交易及收益的詳細檢視。 

[下載付款詳細資料] 會提供付款明細。 如果一項支付已分割為多筆付款，您可以在此詳細檢視中了解其分割方式。
