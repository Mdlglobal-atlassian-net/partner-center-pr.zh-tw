# <a name="core-requirements-for-incentives-co-op-claims"></a>獎勵合作宣告的核心要求

**適用於**

- 合作夥伴中心

您提交獎勵宣告的每個活動類型都有核心要求，您必須符合該核心要求並在您的執行證明文件中證明。 有時候，也需要定義的通訊類型。

下表概述列出每個已接受活動的核心要求和通訊要求。
