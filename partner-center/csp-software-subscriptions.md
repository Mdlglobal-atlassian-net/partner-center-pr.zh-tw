# <a name="sell-software-subscriptions-through-csp"></a>透過雲端解決方案提供者銷售軟體訂閱

**適用於**

- 雲端解決方案提供者

**適當的角色**

- 系統管理代理人
- 全域系統管理員

使用 Azure 保留和伺服器訂用帳戶（Windows Server 和 SQL Server 訂用帳戶），CSP 方案中的合作夥伴可以更有效地解決快速成長的客戶需求，以提供更符合成本效益的解決方案，以支援高預測性和持續性雲端工作負載。 

您現在可以透過合作夥伴中心和 Azure 入口網站，利用 Azure Hybrid Benefit，來代表商業客戶取得、布建及管理 Azure 保留和伺服器訂閱。

Azure Hybrid Benefit 可協助您從您的 Windows Server 授權獲得更多價值，並可節省高達 40% 的虛擬機器。 您可以與軟體保證所涵蓋的 Windows Server Datacenter 和 Standard 版本資料中心授權搭配使用權益。 視版本而定，您可以轉換或重複使用您的授權，在 Azure 中執行 Windows Server 虛擬機器，並支付較低的基礎計算費率（Linux 虛擬機器費率，例如）。

## <a name="buy-software-subscriptions-on-behalf-of-customers"></a>代表客戶購買軟體訂閱

代表客戶購買軟體訂閱：

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

3. 從客戶的詳細資料頁面中，選取 [**新增產品**]，然後遵循畫面上的指示來建立並支付您的訂單。 除了澳大利亞和巴西以外，所有商業定價都不包括稅金。 針對澳大利亞和巴西，價格包含稅金。

## <a name="activate-and-manage-software-subscriptions"></a>啟用及管理軟體訂閱

購買軟體訂用帳戶之後，請遵循下列步驟下載。

>[!NOTE]
>您必須是系統管理員代理程式，才能下載軟體並取得啟用金鑰。

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

3. 移至您客戶的詳細資料頁面，然後選取 **\[軟體\]** 。 您會看到一份代表客戶購買的所有軟體清單。 

4. 展開您要下載的產品。 在 **\[選取產品\]** 欄位中，選取您想要的 **\[版本\]** 、 **\[語言\]** 和 **\[檔案類型/OS\]** 。 

5. 選取 **\[提交\]** 來顯示特定產品。 

6. 選取 **\[取得金鑰和下載項目\]** 。 

7. 選取 **\[下載\]** 以開始下載，或選取 **\[複製連結\]** 複製連結並將其傳送給客戶。 

>[!NOTE]
>此連結會在兩個星期後或 50 次下載後到期 (視何者先發生)。 連結過期之後，請回到此頁面並再次選取 **\[取得金鑰和下載項目\]** 以再啟用另外兩週或 50 次下載。 您可以依需求執行此動作任意次數。 

## <a name="view-activity-for-software-key-access-and-software-downloads"></a>查看軟體金鑰存取和軟體下載的活動

基於審核或合規性目的，您可能需要檢查已存取「伺服器訂閱」軟體金鑰或下載的「伺服器訂閱」軟體的使用者清單。 使用下列程式來存取此資訊。 

>[!NOTE]
>您必須是 [全域管理員]、[帳戶管理員]、[參考管理員] 或 [行銷內容管理員]，才能看到這些活動記錄。 

1. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

2. 選取右上角的齒輪圖示。

3. 在功能表中，選取 [**活動記錄**]。

4. 輸入您想要查看之活動的日期範圍。 [活動記錄檔] 會顯示在您指定的時間內，已存取軟體金鑰或已下載軟體的使用者清單。 

## <a name="cancel-a-purchase"></a>取消購買

您可以在購買日期的60天內取消軟體購買。 如果您在這前60天的期間內取消，將不會向您收取提前終止費的費用。 60天后，您就無法再取消購買。 （請參閱，以瞭解此取消規則的主要限制。 若要瞭解取消軟體購買後會發生什麼事，請參閱這些步驟之後的重要注意事項。） 

>[!NOTE]
>取消購買的下列步驟僅適用于在特定取消視窗中符合取消資格的軟體，例如購買後的前60天內。 這些步驟也不適用於 Azure 中的 SUSE Linux 或 RedHat 軟體方案。 目前，您無法取消或交換 SUSE 或 RedHat 軟體方案。 [深入瞭解](https://docs.microsoft.com/azure/virtual-machines/linux/prepay-suse-software-charges)如何使用 SUSE Linux 或 RedHat 方案。

請遵循下列步驟來取消購買：

>[!NOTE]
>您必須是系統管理代理人才能取消購買。 下列步驟說明如何在合作夥伴中心儀表板中取消購買。 您也可以使用[合作夥伴中心 API](https://docs.microsoft.com/partner-center/develop/cancel-software-purchases)來執行此動作。

1. 在您開始取消程式之前，請確定您具有下列各項：

    - 客戶的名稱、租使用者 GUID 或功能變數名稱

    - 您想要取消之產品的名稱
    
    - 訂單識別碼

2. 登入合作夥伴中心[儀表板](https://partner.microsoft.com/dashboard)。

3. 從 [合作夥伴中心] 功能表上，選取 [客戶]，然後從清單中選擇客戶。

4. 在客戶的詳細資料頁面上，選取 [**軟體**] 以查看為客戶購買的軟體清單。 

5. 找出您想要取消的軟體購買，然後選取 [**取消**]。 隨即會出現一個對話方塊。

6. 從 [訂單號碼] 下拉式清單中，選取您想要取消的正確訂單 ID 編號。 （您可以從客戶的**訂單歷程記錄**頁面瞭解訂單或訂單識別碼的詳細資訊。）

7. 選取核取方塊以確認您已閱讀關於取消的**重要**訊息。 （請參閱下面的**重要**注意事項，以深入瞭解取消購買後會發生什麼事）。

8. 選取 [**提交**] 以取消您的購買。 如果您想要取消客戶的多個訂單，您必須針對每個唯一的訂單 ID 號碼，再次執行步驟4到6。

當您嘗試取消訂單時，合作夥伴中心也可能會提供其他資訊（顯示在 [訂單號碼] 下拉式清單下方）。 這種資訊可以包括：

- 您需要多少天的時間才能取消該特定訂單

- 您是否已通過取消視窗，且無法再取消訂單

- 如果我們需要取消要求的詳細資訊，您可能會獲得**客戶支援要求**表單的連結。

>[!IMPORTANT]
>取消訂單後，將會出現確認取消的訊息。 不過，在 [合作夥伴中心] 儀表板上顯示取消之前，可能會有最多15分鐘的延遲。 另請注意下列的取消後詳細資料。

在您取消購買後：

- 所有相關的軟體金鑰和下載連結都會遭到撤銷。 這表示您或您的客戶都不能使用軟體金鑰，也無法下載與此購買程式相關的連結。 您和您的客戶須負責停所有已取消軟體的使用。 您也必須負責卸載已取消的軟體，並移除任何相關的軟體下載和連結。

- 取消的專案仍會顯示在客戶的軟體詳細資料頁面上，但啟用金鑰將無法使用。

- 已取消訂單的信用額度將會出現在您的下一個月發票上。 永久軟體會收到100% 的點數，而軟體訂用帳戶將會收到按比例計算的信用額度。

### <a name="submit-a-customer-support-request-to-cancel-a-purchase"></a>提交客戶支援要求以取消購買

如果您嘗試取消透過合作夥伴中心購買的軟體，但已告知提供更多資訊並填寫客戶支援要求表單，這些步驟可協助您：

1. 當您從 [取消購買] 視窗選取 [**客戶支援要求**] 連結時，將會開啟 [回報合作夥伴中心問題] 頁面。

2. 在 [詳細資料] 下的 [問題類型] 清單中，選取 [ **CSP 購買/退款代表客戶**]。

3. 填寫 [影響] 和 [標題] 欄位。

4. 在 [描述] 欄位中，提供下列各項：

    - 客戶租用戶 GUID 或網域名稱
    
    - 訂單識別碼或訂閱識別碼
    
    - 退款原因

    - 要求金額

5. 在 [連絡人] 欄位中，輸入您的名稱、電子郵件地址和電話號碼。

6. 如果您因故需附加檔案，請選取 **\[新增檔案\]** 。 這是選用步驟。

7. 當您完成時，請選取 [**提交**]。
