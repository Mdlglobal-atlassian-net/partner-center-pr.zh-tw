# <a name="customer-accounts"></a>客戶帳戶

**適用於**

-  合作夥伴中心
-  美國政府適用的 Microsoft Cloud 合作夥伴中心
-  Microsoft Cloud 德國合作夥伴中心

身為經銷商合作夥伴，您應該要在合作夥伴中心中為每個客戶建立帳戶。 客戶帳戶將會是您銷售訂閱、帳單及提供支援的起點。
