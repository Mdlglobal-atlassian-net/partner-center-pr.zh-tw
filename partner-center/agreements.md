# <a name="microsoft-cloud-agreements-by-region-and-language"></a>依地區和語言的 Microsoft Cloud 合約

**適用於**

- 雲端解決方案提供者合作夥伴

客戶必須先接受並簽署 Microsoft Cloud 合約, 才能代表客戶進行訂單。

>[!IMPORTANT]
> 從2018年11月7日開始, 您必須提供簽署者的相關資訊, 以確認客戶已接受 Microsoft Cloud 合約。 如需詳細資訊, 請參閱[確認客戶接受 Microsoft Cloud 合約](confirm-consent.md)。

Microsoft Cloud 合約是區域特定的, 而且可使用多種語言。 在下列其中一份清單中尋找客戶的位置和慣用語言, 以下載正確的合約。

如果您不確定可以銷售哪些地區和市場, 請在此清單中尋找貴公司的位置:[雲端解決方案提供者方案的地區市場和貨幣](regional-authorization-overview.md)。
