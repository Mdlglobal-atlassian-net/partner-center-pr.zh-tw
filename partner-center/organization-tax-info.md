# <a name="organization-tax-information"></a>組織稅務資訊

**適用於**

-  合作夥伴中心

合作夥伴中心購買項目的稅金將由您的公司地址決定。 某些國家/地區的企業可以提供他們的加值稅 (VAT) 編號或當地的對等項目。

### <a name="add-your-vat-id"></a>新增您的加值稅識別碼

1.  從 **\[儀表板\]** 移至 **\[合作關係\]** -> **\[優惠\]**
2.  按一下 **\[新增您的加值稅識別碼\]**
3.  輸入加值稅識別碼，然後按一下 **\[下一步\]**
